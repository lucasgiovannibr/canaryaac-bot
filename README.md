# CanaryAAC Discord BOT

BOT utilizando API do [CanaryAAC](https://github.com/lucasgiovannibr/canaryaac).


## Instalação

- Utilize PHP 8.1+
- Configure discord.php
    - BOT Token
    - URL CanaryAAC API
- Iniciar BOT utilize o comando

```bash
  php start discord.php
```
